# DESCRIPTION

<h3>This tool will scan for subdomains, and it will also probe for live domains while also retrieving their ip.</h3>

## INSTALLATION

```bash

git clone https://github.com/gotr00t0day/subrecon.git

cd subrecon

pip3 install -r requirements.txt

```

## USAGE

```bash

python3 subrecon.py

```
